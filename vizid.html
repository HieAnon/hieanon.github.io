Visual Identity

This is a page to keep notes and to formalize the visual identity of this portfolio site. It likely won't ever be linked from the index, so if you found it, congratulations, special person!

This site is designed to communication information about me, to highlight selected work in a portfolio, to provide tutorials for various web technologies, etc.

This page will serve as a reference for a consistent brand and visual style.

Typography

This site will typically have the following categories of text markup

  • Major Headers (h1), visually similar across the site to denote the same level of major importance. These are displayed as H1 tags only, in the Display.
  • Subheaders (h2-h6), potentially in different sizes, but maintaining consistent font weight and style. Likely in Rambla or another sans-serif liek DOsis
  • Body text(p), a serifed font. Options: Volkhov, Lora, Georgia, serif
  • Code(.code), displayed by convention in a monospaced font. Specifically, code should be displayed in Inconsolata where possible

Visual Hierarchy

This site will typically have the following categories of text markup

  • Major Headers (h1). Visually distinct/unique on the site. Left-justified, large-ish. With gravitas.
  • Subheaders (h2-h6) maintain consistent font weight and style. Primary differences are size (still larger than body text) and justification. H2 centered, h3 left?
  • Body text(p) should be marked up with a minimum of effort. Left-justified, one-line paragraph breaks, little/no indenting.
  • Code(.code), displayed by convention in a monospaced font, in a shaded block.

Color Palette

There are primary uses for an off-white, an off-black, and three or four accent colors. High contrast should be maintained for accessibility

  • Off-Black: #2a3130
