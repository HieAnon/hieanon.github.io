Visual Identity

This is a page to keep notes and to formalize the visual identity of this portfolio site. It likely won't ever be linked from the index, so if you found it, congratulations, special person!

This site is designed to communication information about me, to highlight selected work in a portfolio, to provide tutorials for various web technologies, etc.

This page will serve as a reference for a consistent brand and visual style.

Typography

This page will typically have the following categories of text markup

  • Major Headers, visually similar across the site to denote the same level of major importance These are displayed as H1 tags only, in Cinzel.
  • Subheaders, potentially in different sizes, but maintaining consistent font weight and style. Likely in Rambla or another sans-serif liek DOsis
  • Body text, a serifed font. Options: Volkhov, Lora, Georgia, serif
  • Code, displayed by convention in a monospaced font. Specifically, code should be displayed in Inconsolata where possible

Color Palette

There are primary uses for an off-white, an off-black, and three or four accent colors. High contrast should be maintained for accessibility

  • Off-white:#dfdbd6
  • Off-Black: #2a3130
  • Header and footer: #439292
  • Accents: #924343, #6b4392, #6a9243
  • consider adapting these colors to some found in htmlcolorcodes.com
