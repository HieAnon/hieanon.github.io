About Me

Hi, I'm Topher! I'm an information designer, librarian, and data geek, tinkering with code to make the world work better.
...at least, that's the plan. I'm still pretty new to web development, so in the meantime, this is a place for me to figure out how to build stuff on the web. (See "Projects" for some things I've been tinkering with lately!)
When I'm not coding, I'm a musician, runner, and occasional baker-of-nomz. Keep an eye out for fun stuff!

About the website

This site was initially created 24 July 2016, and has grown in fits and starts ever since! The full codebase is open source, and on Github--feel free to view the source code or revision history!

I write all of my code with Atom, use SCSS to mark up my stylesheets. I'm inspired by TufteCSS and TabletopWhale, along with loads of other people. I use Cinzel for my headers, and Dosis and Rambla for my other text
